# SDA-TODOLIST-SM

#ToDoList App
A simple JAVA program to create and manage a personal Todo List. 

# Discription 
Simple program for manage your ToDo List by Adding tasks and manage them
by Simple options you can select 1 Choise that help you to use the ToDoList


## features
* Models a task with: 
    * task title
    * due date
    * select project
   
* Display a collection of tasks that can be sorted
    * by date
    * by project
* Support the ability to
    * add task
    * edit task
    * mark a task as done
    * remove (delete) tasks
* Support a text-based user interface
